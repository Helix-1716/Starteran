## 📖 Overview

**StartEarn** is a premium, high-performance web application designed to connect **Students**, **Employees**, and **Recruiters**. Complete with real-time job postings, seamless instant messaging, dynamic dark-mode/luxury visuals, and AI-powered interactions.

Experience a $10M SaaS-grade aesthetic out of the box with beautifully staggered animations, interactive glassmorphism, and neon-accented UI built atop React and modern utility paradigms.

---

## ✨ Features

- **🌐 Multi-Tier User Roles**: Distinct, feature-rich dashboards for **Students**, **Employees**, and **Recruiters**.
- **⚡ Real-Time Job Engine**: Built with Firebase Firestore. Job postings, updates, and deletions reflect instantly across all clients without a page refresh.
- **💬 Instant Messaging**: Seamless real-time chat between users, with read receipts, typing indicators, and instant notification bubbles.
- **🎨 Premium Animations**: Utilizes `Anime.js` for high-end micro-interactions, high-energy hero entrances, responsive dynamic backgrounds (like `WaveNetworkBackground` and `TechGridBackground`), and staggering card reveals.
- **🧠 AI & Ideas Exploration**: Dedicated "Explore AI Tools" and "Explore Ideas" hubs.
- **🏆 Hackathons & Categories**: Discover curated hackathon events and browse beautifully categorized job listings.
- **🛡️ Secure & Scalable**: Authentication and live database handled by Firebase.

## 🛠️ Tech Stack

### Frontend & UI
- **React 18** + **Vite** — Blazing fast component rendering and dev server.
- **TypeScript** — For rock-solid type safety.
- **Tailwind CSS** — Highly customizable utility-first styling.
- **Anime.js** — Fluid physics-based and timeline animations.
- **Lucide React** — Crisp, consistent iconography.

### Backend & Database
- **Firebase Authentication** — Secure user login flows.
- **Firebase Firestore** — Real-time NoSQL database syncing data globally.
- **Firebase Storage** — For avatars, resumes, and media uploads.

## 🚀 Getting Started

Follow these steps to set up the project locally.

### 1. Clone the repository
```bash
git clone https://github.com/your-username/StartEarn.git
cd StartEarn
```

### 2. Install dependencies
```bash
npm install
```

### 3. Set up environment variables
Create a `.env` file in the root directory and add your Firebase configurations:
```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_messaging_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```

### 4. Run the development server
```bash
npm run dev
```
Your app should now be running on [http://localhost:5173](http://localhost:5173).
